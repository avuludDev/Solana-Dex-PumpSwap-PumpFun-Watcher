# Telegram-бот: уведомления о токенах

Отдельный сервис: присылает в Telegram короткие сообщения о **токенах** по многоуровневой фильтрации. WSOL, USDT, USDC не учитываются.

## Многоуровневая фильтрация

| Уровень | Переменная | Описание | По умолчанию |
|--------|------------|----------|--------------|
| 1 | `TELEGRAM_MIN_PURCHASE_USD` | Минимум суммы одной покупки ($) | 500 |
| 2 | `TELEGRAM_MIN_PURCHASES_COUNT` | Один кошелёк купил токен минимум N раз | 3 |
| 3 | `TELEGRAM_MIN_WHALES_COUNT` | Минимум китов (кошельков с 3+ покупками) по токену | 1 |
| 4 | `TELEGRAM_MIN_TOTAL_VOLUME_USD` | Минимальный суммарный объём по токену ($) | 0 |
| 5 | `TELEGRAM_LIQUIDITY_WINDOW_DAYS` | Окно для фильтра ликвидности (1–3 дня) | 2 |
| 6 | `TELEGRAM_MIN_BUY_SELL_RATIO` | Минимум соотношения объём покупок / объём продаж (≥ 1 = набирает) | 1 |

Токен попадает в уведомление только если: 1) прошёл фильтр китов (уровни 1–4); 2) за последние N дней объём покупок по токену ≥ объёма продаж (набирает ликвидность, а не сливают). В сообщении выводится статистика по токену за окно: покупки/продажи в $ и нетто.

## Настройка

1. Создай бота через [@BotFather](https://t.me/BotFather), получи `TELEGRAM_BOT_TOKEN`.
2. Узнай `chat_id` (канал/группа: добавь бота, напиши в чат, открой `https://api.telegram.org/bot<TOKEN>/getUpdates`).
3. В `docker-compose.yml` в сервисе `telegram-bot` задай в `environment`:
   - `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` (обязательно);
   - при необходимости — уровни фильтра (см. таблицу выше) и `TELEGRAM_POLL_INTERVAL` (секунды, по умолчанию 300).

## Запуск

Бот запускается **отдельным контейнером** `solana-telegram-bot`:

```bash
docker compose up -d telegram-bot
```

Или вместе с парсером: `docker compose up -d`.

Состояние «уже отправленных» токенов хранится в `data/telegram_whale_sent.json` (volume `./data:/app/data`).

**Раз в сутки** бот дополнительно отправляет одно сообщение **«Топ-10 по китам»**: токены с наибольшим объёмом покупок от китов (те же критерии: 3+ покупки по 500$+, объём покупок > продаж у кита; WSOL/USDC/USDT не учитываются). Время следующей отправки хранится в `data/telegram_last_digest_ts.txt`.
